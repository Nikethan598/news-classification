NewSense – AI-Based News Article Classification System

Description:

NewSense is a web-based application that uses Machine Learning and OCR (Optical Character Recognition) to automatically classify news articles into predefined categories. The system allows users to either paste text or upload an image of a newspaper article, making it a flexible and real-world solution.

The application extracts text from images using OCR and processes it using an ML model to predict the most relevant category such as **Politics, Sports, Technology, Health, Business**, and more. The goal is to simplify news organization and improve accessibility to categorized information.

Features: 
Classify news using text input
Upload newspaper images for classification
OCR-based text extraction from images
Machine Learning-based prediction
Real-time classification results
Displays predicted category with confidence score
User-friendly web interface


Tech Stack:
Frontend: HTML, CSS, JavaScript
Backend: Python
Machine Learning: Scikit-learn, TF-IDF Vectorizer, Logistic Regression / Naive Bayes
OCR: Tesseract OCR
Libraries: NLTK

Working:
1. User inputs news article (text or image)
2. If image → OCR extracts text
3. Text preprocessing (cleaning, tokenization)
4. Feature extraction using TF-IDF
5. ML model predicts category
6. Output displayed with category and confidence score
